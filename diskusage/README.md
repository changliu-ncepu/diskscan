# DiskUsage

一个高性能的磁盘空间分析工具，使用 Go 编写。支持并发扫描、断点续扫和实时进度显示。

## 功能特性

- **并发扫描**：使用多个 worker 并行计算目录大小，大幅提升扫描速度
- **断点续扫**：扫描中断时自动保存进度，下次运行从断点继续
- **实时进度条**：显示扫描进度、已用时间、预计剩余时间，以及当前正在扫描的目录
- **双格式输出**：扫描结果同时保存为 JSON 和 TXT 两种格式
- **排序展示**：按目录大小降序排列，快速定位占用空间最大的目录

## 配置

在 `main.go` 顶部的配置区域修改参数：

```go
var (
    // 要扫描的根目录
    RootDir = `D:\`

    // 并发 worker 数量
    Workers = 8

    // 仅显示占用最大的 N 个文件夹 (0 = 显示全部)
    TopN = 0

    // 进度文件路径 (断点续扫用)
    ProgressFile = `.scan_progress.json`

    // 结果输出目录
    ResultDir = `.`
)
```

## 使用方法

```bash
cd diskusage
go run main.go
```

## 中断与恢复

扫描过程中按 `Ctrl+C` 中断，进度会自动保存。再次运行时自动加载上次进度，从断点继续扫描。
